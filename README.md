# Version 1.0.1
A blacklist and whitelist discord bot made with discord.js v12 & quick.db v7.1.1

# 📝 Set-Up
- ```npm i```
- ```npm start```

Commands | Usage
------------ | -------------
whitelist | whitelist <@user> <-- Also supports user id
blacklist | blacklist <@user> <-- Also supports user id
